파이썬 (.py) 파일을 생성하거나 수정할 때는 반드시 다음 규칙을 준수해야 합니다.

1. **상단 Docstring 필수**: 모든 파이썬 파일의 최상단에는 해당 파일의 목적, 주요 기능 등을 포함하는 docstring(모듈 수준의 주석)을 반드시 작성해야 합니다.
2. **언어**: 모든 주석과 설명은 한국어로 작성합니다.
3. **형식**: 파일의 가장 첫 번째 줄부터 시작하며, 삼중 따옴표(`"""`)를 사용합니다.
4. **포함 내용**:
   - 파일의 목적 및 개요
   - 주요 구성 요소 (클래스, 함수 등)에 대한 간략한 설명
   - 작성 시점의 맥락이나 주의사항 (필요한 경우)

**예시:**

```python
"""
NPS(국민연금) 사업장 데이터 분석을 위한 전처리 스크립트입니다.

주요 기능:
- CSV 파일 로드 및 결측치 처리
- 업종별 평균 급여 추정 로직 포함
- 분석용 정제 데이터 저장
"""

import pandas as pd
import numpy as np

# 코드 시작...
```
